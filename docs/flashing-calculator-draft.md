## הרעיון בלב
המטרה: **שיבינו את העניין** — שמילה אחת היא לא מספר אחד, אלא **שמונה פנים**.
היום ההצלבות הן רשימה סטטית. החזון: **מחשבון חי שמלמד את הקונספט בתנועה**.

## איך זה מרגיש (התנהגות)
1. **נכנסים לדף ההצלבות** → ליד התוכן יש מחשבון גימטריה תמידי (sidebar/צמוד).
2. מספר על המסך **מהבהב** — מושך את העין — מתחיל ב**רגיל**.
3. במחשבון רואים את אותו ערך **"יורד"/זורם** משיטה לשיטה:
   רגיל → מילוי → (שיטה הבאה) → ... **הלוך וחזור**, אנימציה מתמשכת.
4. ההבהוב/הזרימה ממשיכים **עד שנוגעים** — אז זה **ננעל/מתמקד** על אותו מספר,
   והמחשבון "תופס" אותו ומראה את כל 8 השיטות + ההצלבות שלו.

## למה זה חזק
- **מלמד בלי להסביר** — העין רואה את אותו מספר משנה צורה בין השיטות, ומבינה
  לבד ש"רגיל ומילוי ומסתתר הם אותו דבר ממבטים שונים".
- **מזמין מגע** — ההבהוב אומר "גע בי", והמגע נותן שליטה (עוצר את הריצה).
- **מחבר הכל** — המחשבון + ההצלבות + (בהמשך) מנוע הסיפור הופכים ליחידה אחת.

## רכיבים שכבר קיימים לבנות עליהם
