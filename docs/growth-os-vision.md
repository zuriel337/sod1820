# 🚀 Growth OS של Sod1820
## מערכת הצמיחה האוטונומית של «קוד המציאות»

> המטרה אינה רק לפרסם יותר תוכן.
> המטרה היא לבנות מנוע ש**מזהה, מפיץ, מודד ומשפר את עצמו** — כך שכל גילוי, הצלבה או תובנה
> שנוצרים במערכת יוכלו להגיע **לקהל הנכון, בזמן הנכון, בערוץ הנכון**.

---

## 🏛️ שכבה 0 — התשתית
- **Meta App + System User Token** — מפתח־העל לכל שירותי Meta.
- **Supabase Edge Functions** — מוח האוטומציה של המערכת.
- **Vault מאובטח** — ניהול סודות, טוקנים והרשאות (לעולם לא דרך הצ'אט).

## ⭐ שכבה 1 — מעקב אמת (Server-Side)
**Conversions API** — כל פעולה משמעותית באתר נמדדת ישירות מהשרת:
צפייה בדף · שימוש במחשבון · קריאת פוסט · חיפוש ELS · הרשמה · שיתוף · מסע שהתכנס.

**Advanced Matching** — זיהוי מדויק יותר של משתמשים באמצעות נתונים מגובבים (Hash).

> 🎯 המטרה: כמעט **100% קליטת אירועים**, גם כשדפדפנים חוסמים עוגיות.

## 📡 שכבה 2 — הפצת תוכן אוטומטית
- **Facebook** — פרסום אוטומטי · תגובה ראשונה עם קישור · UTM Tracking.
- **Instagram** — פוסטים · Reels · סדרות תוכן.
- **ניתוב מותגים** — כל תוכן מסווג אוטומטית:
  - **קוד המציאות** → תוכן חקירתי, הצלבות, גילויים ונתונים.
  - **כי לה' המלוכה** → תוכן רוחני, גאולה, תורה ותודעה.
  - המערכת מחליטה לאיזה ערוץ התוכן יופץ.

## 🎯 שכבה 3 — קהלים תכנותיים
יצירה אוטומטית של **Custom Audiences** + **Lookalike Audiences**, ללא עבודה ידנית.
דוגמאות: השתמש במחשבון · קרא 3+ מאמרים · ביצע חיפוש ELS · השלים מסע התכנסות.
> כל קהל הופך אוטומטית לבסיס להרחבת החשיפה.

## 📱 שכבה 4 — WhatsApp Growth Engine
המטרה אינה קבוצות. המטרה היא **נכס בבעלות מלאה**.
- **הרשמה** — "🔔 קבל עדכוני קוד המציאות בוואטסאפ".
- **שידורים** — פוסט חדש · גילוי חדש · הצלבה חדשה · מסע מיוחד.
- **בוט אינטראקטיבי** — משתמש שולח שם / מספר / מילה / פסוק → ומקבל תשובה אוטומטית.
- **Click-to-WhatsApp** — מודעות שמביאות משתמש ישירות לשיחה.

## 💰 שכבה 5 — אוטומציית קמפיינים
כאשר יופעל תקציב: יצירת קמפיינים · יצירת קבוצות מודעות · אופטימיזציה ·
עצירת מודעות חלשות · הגדלת תקציב למודעות מנצחות — באופן אוטומטי.

## 📊 שכבה 6 — מרכז הבקרה
מסך ניהול אחד שמציג: איזה מותג מוביל · איזה ערוץ מנצח · איזה פוסט מביא תנועה ·
איזה גילוי יוצר המרות · איזה מסע שותף הכי הרבה. כל הנתונים במקום אחד.

```
יצירה → הפצה → מדידה → למידה → שיפור → יצירה חדשה → ↺
```

## החזון
Sod1820 אינו רק מאגר גימטריה. הוא מערכת ש**מגלה קשרים, בונה תובנות, מפיצה אותן לקהלים
המתאימים, ומפתחת את עצמה** באמצעות הנתונים שהיא מייצרת.
זהו השלב שבו «קוד המציאות» הופך ממנוע חיפוש — ל**מערכת אקולוגית שלמה**.
