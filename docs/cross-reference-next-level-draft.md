## איך הכל מתחבר
**הצלבות + נדירות + מנוע הסיפור + המחשבון המהבהב** = יחידה אחת חיה:
נכנסים → מספר מהבהב → המחשבון זורם בין השיטות → נוגעים → נפתחות ההצלבות
עם דרגת נדירות → "למה מעניין?" → מסר AI → ואם זה אוצר, 💎.

## פתוח / הערות נתונים
- "הצלבת השבוע" צריך קריטריון אוטומטי ל"מפתיע" (נדירות גבוהה × משמעות × טריות).
- ניקוי: זוגות שנבדלים רק בגרש (למשל «כי לה המלוכה» מול «כי לה' המלוכה») נספרים כשניים — לאחד.
- רוב ה-🔴 הם בשיטת קדמי (ערכים גדולים → טבעית נדירים); אפשר לשקלל נדירות לפי השיטה.
