# cross-card — גנרטור כרטיסי הצלבות לשיתוף

Edge Function שמייצר **כרטיס PNG ממותג** (שפת ההיכל) לחידוש הצלבה/גימטריה — יחידה ויראלית מוכנה לשיתוף בפייסבוק/וואטסאפ.

## מה הוא עושה
- קלט: `POST { id }` (uuid מ-`insights`) → שולף כותרת + `related_numbers[0]` + `related_phrases`.
  או `POST { number, title, subtitle }` לכרטיס חופשי.
- מרנדר כרטיס 1080×1080: רקע כהה `#070506`, מסגרת זהב, **המספר גדול בצהוב** (Frank Ruhl Libre 900), כותרת החידוש, הביטויים, חותם `✓ מאומת במנוע`, לוגו `♛ סוד 1820`, `כי לה׳ המלוכה`, וכתובת `sod1820.co.il` מוטבעת (כך המותג נוסע עם התמונה גם בלי לינק).
- מעלה ל-Storage `gallery/cards/` ומחזיר `{ url }`. אם הגיע `id` — מעדכן `insights.card_url`.

טכנולוגיה: `satori` (HTML→SVG, תומך RTL/בידי עם פונט) + `@resvg/resvg-wasm` (SVG→PNG). פונטים עבריים (Frank Ruhl Libre + Heebo) נטענים מ-fontsource.

## פריסה (צריך אישור)
פעולת `deploy_edge_function` דורשת אישור. לאחר הפריסה:
```
# בדיקה ויזואלית — יוצר כרטיס לחידוש קיים ומחזיר url
curl -X POST https://linswmnnkjxvweumprav.supabase.co/functions/v1/cross-card \
  -H "Authorization: Bearer <ANON/JWT>" -H "Content-Type: application/json" \
  -d '{"id":"<insight-uuid>"}'
```
מומלץ לאחר פריסה ראשונה: ליצור כרטיס-דוגמה, להציג, ולכוונן גדלים/צבעים לפי הצורך.

## חיבור לשיתוף פייסבוק
הכרטיס (`card_url`) הוא ה-`image_url` האידיאלי לפוסט פייסבוק של חידוש. בשילוב `share-to-facebook`:
- פוסט-תמונה (הכרטיס) + כיתוב נקי + **לינק בתגובה הראשונה** (`FB_LINK_IN_COMMENT`, ברירת מחדל פעיל) — שומר על reach.
- כל לינק נושא UTM (`FB_UTM`) למדידת תנועה מפייסבוק.

## הרחבות אפשריות (עתידי)
- כפתור "צור כרטיס + שתף" בפאנל הניהול / בבית המדרש.
- וריאנט פורטרט 1080×1350 ו-OG 1200×630.
- כרטיס מרובה-שיטות (DNA המספר: רגיל/מסתתר/קדמי/מילוי בשורה).
