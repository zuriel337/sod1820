# טיוטה: «הצופן בקולנוע: האם משחקי הדיונון הם מפת גאולה נסתרת?» 🦑

**סטטוס:** טיוטה בלבד — **לא פורסם** (לבקשת צוריאל). לא הוכנס ל-DB.

## קבצים
- `full-content.html` — תוכן הפוסט (בטוח-לפרודקשן: inline styles בלבד, בלי `<style>`, בלי inline-height, גרפיקות כ-`<img>`, מילים עם `data-gem` לפתיחת מחשבון אינליין).
- `hero.svg` — גרפיקת הירו (כותרת + 4 המספרים: 456/067/001/218).
- `pyramid.svg` — גרפיקת פירמידת הציות ○△□.
- `preview-full.png` — צילום רינדור מלא בסימולציית פרודקשן (עם כלל ה-WP height-crush) — אומת ויזואלית.

## גימטריה — הכל אומת במנוע הרשמי (רגיל)
| ביטוי | ערך | מקור בסרט |
|---|---|---|
| דיונון = עבדים | 126 | המילה «דיונון» |
| בינה = ניאו | 67 | שחקן 067 (סה-ביוק) · גשר למטריקס |
| ירח = זוהר | 218 | שחקן 218 (סנג-וו) · ירח=מלכות דוד/לבנה (פרשנות) |
| פרצוף | 456 | שחקנים/פרס/זוכה |
| א | 1 | שחקן 001 (איל-נאם, הבורא) |

הערה: **לבנה=87, לא 218** — מוטיב «מלכות דוד נמשלה ללבנה» הוא **פרשני** (לא משוואה). החיבור המספרי האמיתי הוא ירח=זוהר=218.

## איך לפרסם (כשצוריאל יאמר «תעלה»)
1. להעלות `hero.svg` ו-`pyramid.svg` ל-bucket `gallery` בנתיב `posts/squid-game-*.svg` (Storage REST, anon publishable key) ולקבל URLs ציבוריים.
2. להחליף ב-`full-content.html` את `src="hero.svg"` ו-`src="pyramid.svg"` ב-URLs הציבוריים.
3. `insert into posts (...) overriding system value values (...)` — `id`/`wp_id` מעל ה-max; `source='ai'`; `theme='legacy-dark'`; קטגוריה **«הצופן בקולנוע»** + תיוג **«מימד חמש»**; `image_url` = URL ההירו; `modified=date` (לא null).
4. **`ai_touched=true`** — כדי לרנדר את חותמת «🔵 AI · מאומת» בראש (יש כאן ≥2 התכנסויות מאומתות: 67=בינה=ניאו, 218=ירח=זוהר, 126=דיונון=עבדים → מצדיק את הריבוע והחותמת).
5. אימות אחרי INSERT: `select id,slug,date,modified from posts where slug='squid-game-456' limit 1;` — אם modified=null → לעדכן.
