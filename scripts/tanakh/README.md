# בסיס-נתוני התנ"ך המלא

מקור: **MAM — מקרא על פי המסורה (כתר ארם-צובא)** דרך Sefaria API — אותו מקור כקבצי-התורה הקיימים.

## פלט
- `public/tanakh-verses.json` — `{books[39], method:"רגיל", bookRanges[[from,to]], verses[[bookIdx,ch,vs,text,gematria]]}` · 23,204 פסוקים.
- `public/tanakh-letters.txt` — אותיות-עיצורים רצופות (סופיות מנורמלות, בלי רווחים) לדילוגי-אותיות · 1,204,583 אותיות.

## חוק-קדושה
**התורה (5 ספרים) נשארת קנונית — 304,805 אותיות בדיוק.** חילוץ-MAM טרי של התורה נותן 306,269 (פער ~1,464, כנראה כתיב/קרי), לכן הצינור **משתמש ב-`torah-letters.txt`/`torah-verses.json` הקיימים כפי-שהם**, ומוסיף רק נביאים+כתובים מהחילוץ.

## הרצה
1. `python3 scripts/tanakh/gen_tanakh.py`   — מושך את כל 39 יחידות-הספר → קבצי-תנ"ך זמניים.
2. `python3 scripts/tanakh/combine_tanakh.py` — מחליף את חלק-התורה בקנוני, כותב את הקבצים הסופיים.
